Naraku Shonin is a bot developed to provide useful features for the Naraku Mercantile. It's very early in it's life, but we're excited to see it grow!

See below for a preview of what features we have in mind!

We'll add the ability to define and store currencies, and implement a few basic gamba games.

Eventual plans include a Dalamud plugin that can help dealers and players play those games in-game.

We're going to support automating repetitive discord tasks, to make things easier on server moderators.

We're going to integrate a ticket bot to make communication between officers and members easier. Why not use one of the many others out there? Because Discord bots can't call each other, so if we want to be able to automate things involving tickets in the Automation module, we need to have our own ticket bot.

We're going to support custom commands. Because why not?
